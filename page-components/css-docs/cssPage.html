css 문법을 사용하는 세가지 방법

  • 1. html 태그 안에 style="속성:속성값"을 직접 입력
  • 2. head 안에 style 태그를 넣고 안에다가 한꺼번에 정리
  • 3. css 문서를 따로 작성해서 사용

2, 3의 경우 태그명{속성:속성값; 속성:속성값;}과 같은 형태로 사용

class = . = 그루핑하다

html 태그에 class=[00]를 부여해서 style에서 사용할 수 있다

id = #, id>class>tag 선택자의 우선순위를 가진다.

콤마를 사용해서 동시에 같은 태그에 같은 스타일 설정을 할 수 있다.

박스 모델

태그는 인라인태그(inline elment)와 블록태그(block level element)가 있다

display속성을 사용해서 인라인<->블록을 바꿀 수 있다

padding속성 : 컨텐츠와 테두리(border) 사이

margin속성 : 테두리와 다른 element의 테두리 사이

width, height 속성으로 element 자체의 크기를 지정할 수 있다.

그리드(grid)

div태그를 사용하고 id를 주어서 스타일링 할 수 있다.

grid-template-columns:100px 1fr과 같이 내용을 쓰면 첫 element는 100px을 쓰고 뒤의 element는 나머지 공간 다 쓴다.
